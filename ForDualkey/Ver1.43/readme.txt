●Ver1.43 (DualKey) の変更点
通常版(ForM5StackLite)のクライアントから分岐し、M5ATOM S3 Lite + 縦振り/複式パドル対応にしました。
・パドル2本足(DIT/DAH)によるイアンビック・キーヤー(Mode A, スクイズ運用対応)を内蔵
・WPM・パドル左右入替をAndroidアプリから送信して即時反映(EEPROM保存)
・USB-CDC(Android有線)/USB-NCM(iOS USB-C直結)/BLE NUS(Android無線)の3モードに対応
・WiFiスタンドアロンモードも従来通り利用可能(プロファイルはEEPROM保存、SDカード不要)
・サーバーとの通信プロトコルは通常版Ver1.43と共通のため、サーバー機は流用可能

●ファームウェアファイル

Serv_M5AtomLitev1.43.bin  : サーバー用 M5ATOM Lite (通常版Ver1.43と同一のものを同梱)

●初期設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
M5Burnerでバイナリファイルを取り込んでください。
本バージョンはWiFiプロファイルをEEPROMで管理するため、SDカードは不要です。
初回起動時は fallbackProfile (SSID: RemoteKeyer-AP) でUSBモード起動します。
WiFiプロファイルを変更する場合はソースコードを書き換えて再ビルドしてください。

●ソフトウェアについて
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
コンパイルはVisual Studio Code + PlatformIOで行ってください。
クライアント用プロジェクトを作成し、src/main.cppにソースコードをコピーしてください。
platformio.iniは添付のものを使ってください(board = m5stack-atoms3)。
サーバーは通常版(ForM5StackLite)のソースコード・ファームウェアと共通です。

なお、本家のリモートキーヤーキット for Arduino UNO R3とソフトウェア互換性はありませんので、
本家クライアント/サーバーとの通信はできません。
